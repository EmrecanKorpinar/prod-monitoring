## Problem
Production servers fail due to lack of monitoring, alert spam and security visibility.

## Solution
A production-grade monitoring and alerting system with load balancing and security checks.

## Architecture
- NGINX Reverse Proxy & Load Balancer
- Multiple backend instances
- Bash-based metrics & security scanning
- Alert flood prevention
- Dockerized environment

## Tech Stack
Linux, Bash, Docker, NGINX, Node.js

## How to Run
docker-compose up -d
